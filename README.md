# PhiCortex
PhiCortex is a machine learning-based phishing website detection system designed to identify malicious websites using intelligent analysis of URL and domain features. The system extracts structural and lexical characteristics from website links, performs feature engineering and selection, and trains supervised learning models such as Random Forest, Support Vector Machine, and Logistic Regression to classify websites as phishing or legitimate. The project includes model comparison, performance evaluation using standard metrics (accuracy, precision, recall, F1-score, ROC-AUC), and feature importance analysis to understand decision behavior. The objective is to build a robust, data-driven detection framework that enhances web security through intelligent pattern recognition rather than traditional rule-based methods.
